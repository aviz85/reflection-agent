# Yuval Pipeline

Full post-session pipeline: merge Zoom recordings → transcribe → format summary → send via WhatsApp.

## Usage

```
/yuval-pipeline [recording-path-or-dir]
```

If no path given, scans `~/Downloads` and `~/Documents/Zoom` for today's recordings.

## Steps

1. **Find recordings** — locate `.mp4`/`.m4a` files from today's Zoom session
2. **Merge** — if multiple parts, merge with FFmpeg into single file
3. **Transcribe** — use `transcribe` skill (ElevenLabs Scribe v2 default, `--cheap` for Groq)
4. **Summarize** — extract: key topics, decisions, action items, quotes
5. **Format** — Hebrew, short paragraphs, phone-readable
6. **Send** — WhatsApp via `whatsapp` skill to יובל (number from contacts via `get-contact`)

## Options

- `--no-send` — stop after summary, don't WhatsApp
- `--cheap` — use Groq for transcription
- `--to PHONE` — override recipient

## Dependencies

- `transcribe` skill
- `whatsapp` skill  
- `get-contact` skill
- FFmpeg (for merge)
