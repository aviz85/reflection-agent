# Post-Meeting Pipeline

After every Zoom call, run this skill to close the loop automatically.

## Trigger
```
/post-meeting [recording-path] [--wa] [--lang he|en]
```

## Pipeline

1. **Merge** — if multiple recording segments exist, concatenate with FFmpeg
2. **Transcribe** — ElevenLabs Scribe v2 (default) or `--cheap` for Groq
3. **Summarize** — Claude: extract decisions, action items, open questions (Hebrew by default)
4. **Deliver** — `--wa` sends summary + transcript link via WhatsApp to Aviz

## Output Files (saved to `~/meetings/YYYY-MM-DD/`)
- `transcript.srt` — full timestamped transcript
- `summary.md` — decisions + action items + open questions
- `recording-merged.mp4` — merged video (if segments existed)

## Dependencies
- `transcribe` skill
- `whatsapp` skill
- FFmpeg

## Notes
- Default language: Hebrew
- Always saves locally even without `--wa`
- If no path given, scans `~/Downloads/` and `~/Desktop/` for recent `.mp4`/`.m4a` files
